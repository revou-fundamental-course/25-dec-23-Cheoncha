SEGITIGA

Rumus Luas Segitiga yaitu:
L=1/2 x a x t
Di mana:
L = Luas
a = Panjang Alas
t = Tinggi

Rumus Keliling Segitiga yaitu:
K= S1 + S2 + S3
Di mana:
K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C

JAJAR GENJANG

Rumus Luas Jajar Genjang yaitu:
L= a x t
Di mana:
L = Luas
a = Panjang Alas
t = Tinggi

Rumus Keliling Jajar Genjang yaitu:
K= 2 x (AB+BC)
Di mana:
K = Keliling
AB = CD = Sisi Datar
BC = AD = Sisi Miring
